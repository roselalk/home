At the start of 2022, I ventured into the art of web development for the first time.

I had gained some knowledge about HTML, CSS and JavaScript before, but had used it mostly to make gimmicks like a green screen simulator.

However, for my application to Make IT Work I wanted to be prepared, and I wanted to show what I'd learned. So, I took a course and started learning proper web development, so I could do just that.

My first website started out pretty basic.

But I was proud! My first website! And when I learned that even the world's greatest Computer Science professors had webpages similar to mine, surely it had to be good enough for me too, right?

But I enjoyed it so much, that I kept improving it...

And improving it...

Until I was satisfied.
